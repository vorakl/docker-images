A collection of useful docker images

A collection of useful docker images. For all images, once a day and on each commit to this source repository, automatically are being run:

  • tests
  • builds

For testing services in containers is being used this technic.

Available images

Any of these images can be downloaded by specifying a name[:tag]. For example docker pull vorakl/centos:latest or docker pull vorakl/alpine.

  • Base images
    • vorakl/centos
    • vorakl/alpine
  • Service images
    • vorakl/alpine-pelican
    • vorakl/centos-opensmtpd
